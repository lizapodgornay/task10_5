# task10_5
Выполняемый анализ
Среднее количество просмотренных страниц - вычисляет среднее арифметическое по всем визитам

Стандартное отклонение времени на сайте - измеряет разброс длительности сессий

Коэффициент вариации - оценивает изменчивость времени на сайте:

0.25 → Высокая вариабельность

0.1-0.25 → Умеренная вариабельность

< 0.1 → Слабая вариабельность

Сравнение поведения на разных устройствах - проверяет статистическую значимость различий в количестве просмотренных страниц между мобильными и десктопными пользователями с помощью t-теста Стьюдента:

p-value > 0.05 → есть статистически значимая разница

p-value ≤ 0.05 → нет статистически значимой разницы

Полученные метрики могут использоваться для:

Оценки средней вовлечённости пользователей

Понимания стабильности пользовательского поведения

Принятия решений об адаптации контента для разных типов устройств

Выявления областей для улучшения пользовательского опыта
